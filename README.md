# 2048

### [Play (Phone friendly)](https://mira1zu.github.io/game_2048/)

## Technologies used:

- **TypeScript**
- **React** - UI library
- **Redux Toolkit** - state management
- **SCSS** - styling
- **Lodash** - miscellaneous

## Running locally:

- Clone this repo
- `npm install`
- `npm start`
- open your browser at `http://localhost:3000`
